World Clock

World Clock designed with HTML, CSS and JavaScript with API integration using moment.js.com. Select a city from the drop-down menu to see current time or select current location. World Clock displays current time as well as current date. Background changes when selecting from the drop-down menu. Hosted on Netlify and open-sourced on GitHub.

Weather App

Weather App designed with HTML, CSS and JavaScript with API integration using OpenWeatherMap.org. Application lists current temperature, humidity and windspeed. Weather icons correspond with current weather. Five-day forecast also listed on weather application. Application has function to search a city or current location. Application displays current date and time. Hosted on SheCodes platform.
